# **App Name**: Ivry Pharma Digital

## Core Features:

- Product Catalog: Browse and search for products including parapharmacy, supplements, and medical equipment.
- User Accounts: Secure user accounts with different roles: Admin, Collaborator, and Client, each with tailored access and permissions.
- Admin Dashboard: Comprehensive dashboard for store management, stock control, user management, and blog management. Annuaire to manage partnerships
- Collaborator Dashboard: Limited access dashboard for stock management, product editing, and draft blog creation.
- Client Dashboard: Dashboard for clients to manage their profile, orders, messages, and video appointments.
- AI Chat Assistant: An AI-powered chat tool to assist customers with product selection, information, and support, escalating to a pharmacist when necessary.
- Stock Management: An AI tool alerts the admin if a stock is below threshold to facilitate and optimize smart inventory.
